# 赤壁之战 · 探索课堂

一个面向小学生的探究式互动历史课件，基于 [visual-cognition-slides](https://github.com/Lydia2214/visual-cognition-slides) 设计哲学构建。

## 特色

- **探究式学习**：先抛出谜题/线索，让学生猜测，再逐步揭示答案
- **10 张交互幻灯片**，覆盖赤壁之战完整故事线
- **逐步触发**：所有内容由学生主动点击解锁，无自动播放
- **纯单文件 HTML**，无需服务器，直接浏览器打开即可使用

## 幻灯片结构

| 编号 | 主题 | 交互方式 |
|------|------|----------|
| S1 | 封面：5万 VS 80万 | 点击开始 |
| S2 | 三方兵力谜题 | 点击3张卡片揭示兵力 |
| S-MAP | 地理探索地图 | 点击4个营地了解地理优势 |
| S3 | 草船借箭谜题 | 4选1猜测 |
| S4 | 草船借箭全过程 | 逐步点击揭示5步时间轴 |
| S5 | 铁链连船：好主意？ | 二选一判断 |
| S6 | 火攻三要素翻牌 | 翻开3张卡牌 |
| S7 | 多米诺连锁反应 | 点击触发因果链 |
| S8 | 历史小侦探测验 | 3道选择题 |
| S9 | 总结口诀 | 打字机动画 + 知识地图 |

## 使用方式

直接用浏览器打开 `index.html`，支持：
- 鼠标点击 / 空格键：前进
- 方向键左右：前进 / 后退
- 触屏左右滑动：切换幻灯片
- `F` 键：全屏

## 技术栈

纯原生 HTML + CSS + JavaScript，零依赖，单文件，67KB。
