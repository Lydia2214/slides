## 数据格式说明

每个 `.json` 文件包含完整的课堂结构：

```json
{
  "classroom": {
    "id": "课堂ID",
    "stage": { "name": "课堂名称", "language": "zh-CN", ... },
    "scenes": [ { "type": "slide|quiz|interactive|pbl", "title": "...", "content": {...} } ]
  }
}
```

可通过 OpenMAIC API 重新导入：
```bash
curl -X POST https://open.maic.chat/api/classroom \
  -H "Content-Type: application/json" \
  -d @hQVpWTAoQd.json
```
